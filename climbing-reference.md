# Climbing Domain Reference

Interpretive context for reading climb/session notes correctly. This is
separate from Project Instructions (which cover data access, schema, and
display rules) — this file covers venue-specific conventions, grading
systems, and terminology that don't live in the database itself.

## Peak Boulders — Grading

Peak Boulders uses color-coded tags for grade-range circuits rather than
single-grade labels. When a climb or note references a tag color, translate
it to the corresponding V-range:

| Tag    | Grade range |
|--------|-------------|
| Green  | V0–V1       |
| Yellow | V2–V4       |
| Orange | V4–V6       |
| Red    | V6–V8       |
| Purple | V8–V10      |

## Triangle Boulders — Setters

Climbs at Triangle Boulders are commonly referred to by their setter
("author") rather than by name alone. When a note says "[Setter] climb,"
it means "the climb set by [Setter]."

Example: *"This Augie climb was hard"* = "This climb, set by Augie, was hard."

Known setters:
- Augie
- TC / Tyler
- MD
- IA
- Ken
- Brian

Note: TC and Tyler refer to the same setter — treat as equivalent when
matching notes.
